Projetos

Projeto - Currículo

Meu primeiro projeto é uma representação inicial do meu percurso na área de linguagem da programação. Com um layout simples, ele foi concebido com o objetivo principal de adquirir uma compreensão dos conceitos fundamentais que sustentam a construção de websites.Este projeto me permitiu aprender como selecionar e aplicar cores de forma eficaz, contribuindo para uma estética visual atraente. Além de, adquiri a habilidade de criar links que direcionam os usuários para diferentes sites. E dividir o conteúdo em seções distintas, aprendi a organizar informações de maneira lógica e facilitar a acessibilidade. A criação de parágrafos, listas e a estruturação adequada do conteúdo foram pontos essenciais do projeto.
Este projeto marcou o início da minha jornada, fornecendo uma base sólida para explorar conceitos mais avançados. Embora tenha um layout simples, sua importância é inestimável, pois me deu a base necessária para iniciar e aprofundar o meu conhecimento e habilidades.

Projeto - Cardápio

No meu segundo projeto, já com o Git instalado e uma base sólida de conhecimentos iniciais em desenvolvimento web, minha missão era aprofundar ainda mais meu entendimento sobre o design de páginas da web. Este projeto representou uma oportunidade empolgante para explorar os aspectos visuais e criativos do desenvolvimento web.
Nesse projeto tive a oportunidade de experimentar diferentes propriedades de exibição (display) em CSS. Isso me permitiu compreender como controlar a disposição dos elementos na página, criando layouts flexíveis e adaptáveis.Compreendi como criar colunas e organizar o conteúdo de forma dinâmica, melhorando a usabilidade e a estética das páginas. Ademais, também fui capaz de incorporar novas fontes tipográficas, introduzir elementos como imagens estáticas e animadas e explorar uma variedade de recursos visuais para tornar as páginas mais envolventes.A parte mais empolgante deste projeto foi a liberdade de criar um tema personalizado e construir uma página temática.

Projeto - ODS

No meu terceiro projeto, direcionei minha atenção para a estilização do fundo e das bordas de páginas da web. Esta etapa do meu percurso de desenvolvimento web me permitiu aprofundar meu conhecimento em aspectos visuais e estéticos do design web.A estilização do background e das bordas desempenha um papel crucial na construção de páginas que não apenas informam, essas habilidades ajudam a entender o que chama atenção de um usuario em uma página de internet. Ao mesmo tempo que me aproximou ainda mais dos temas da minha pesquisa em pedagogia e tecnologia digital. Aprendi como utilizar gradientes, imagens de fundo e cores para criar atmosferas visuais que complementam a narrativa e o conteúdo das páginas.
Foi bastante significativo trabalhar em um tema que se alinha estreitamente com minha linha de pesquisa. Foi bastante prazeroso juntar esses dois mundos.
